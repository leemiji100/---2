#### 🖼️ 사용 도구 ① — 이미지 생성 : Midjourney

**목표**
한복의 질감과 전통문양의 디테일이 살아있는 고화질 키 비주얼 생성

**프롬프트 원문 (English)**
```
Cinematic full shot of a person wearing a luxurious Hanbok
with intricate Joseon traditional patterns, spinning elegantly
in a traditional Korean palace courtyard, Dancheong patterns
on the eaves, soft golden hour lighting, silk texture detail,
hyper-realistic, 8k, shallow depth of field,
vibrant colors --ar 16:9 --v 6.0
```

**키워드 설명**

| 키워드 | 역할 |
|--------|------|
| `intricate Joseon traditional patterns` | 복잡하고 정교한 조선 전통문양 강조 |
| `spinning elegantly` | 인물의 회전 동작 암시 → Runway 변환 시 유리 |
| `Dancheong patterns` | 배경(궁궐 처마)에도 문양을 넣어 세계관 확장 |
| `--ar 16:9` | 영상 규격에 맞는 가로 비율 설정 |

#### 🎥 사용 도구 ② — 비디오 변환 : 드롭샷

**목표**
정적인 이미지를 인물이 회전하며 옷자락이 휘날리는 역동적인 영상으로 변환

**프롬프트 원문 (English)**
```
The person in Hanbok is slowly spinning in a 360-degree circle,
the fabric of the Hanbok with traditional patterns is fluttering
elegantly, soft camera orbit movement,
cinematic slow motion, high quality.
```

회전이 마무리되는 단계, 고궁의 목조 기둥 사이로 긴 그림자가 드리워진 배경에서 한복을 입은 인물이 한 바퀴를 돌아 다시 정면을 향해 멈춰 서 있다. 휘날리던 치맛자락은 공중에서 서서히 아래로 떨어지며 겹겹이 쌓이는 모양새다. 카메라는 인물을 아래에서 위로 올려다보는 로우 앵글 풀샷으로 고정되어 있다. 한복을 입은 인물이 회전의 관성을 죽이며 멈추자 치맛자락이 시네마틱 슬로우 모션으로 찰랑거리며 다리 주변으로 부드럽게 내려앉는다.



**키워드 설명**

| 키워드 | 역할 |
|--------|------|
| `slowly spinning in a 360-degree circle` | 인물의 회전 동작을 구체적으로 명령 |
| `fabric is fluttering` | 한복 특유의 선과 문양이 잘 보이도록 옷자락 움직임 강조 |
| `camera orbit movement` | 카메라가 인물을 따라 도는 입체감 부여 |

#### 🎵 사용 도구 ③ — 오디오 생성 : Suno AI

**목표**
전통의 무게감과 현대적인 세련미가 공존하는 배경음악 생성

**Style Description**
```
Korean traditional fusion, upbeat and bouncy Gayageum melody,
rhythmic Samulnori percussion, Janggu, Kkwaenggwari,
cinematic, elegant, high energy
```

**Lyrics (Custom Mode)**
```
조선의 문양을 입고 태어난 우리의 전통문양.
(또는 Instrumental)
```

**키워드 설명**

| 키워드 | 역할 |
|--------|------|
| `bouncy Gayageum` | 통통 튀는 가야금 소리로 트렌디한 느낌 부여 |
| `Samulnori percussion` | 사물놀이 타악기로 리듬감과 에너지 추가 |
| `fusion` | 올드하지 않게 현대적인 감각을 혼합 |
